# Application d'Orthophonie IA

Une application web pour améliorer la diction avec l'aide de l'intelligence artificielle.

## Fonctionnalités MVP
- Enregistrement audio via microphone
- Reconnaissance vocale en français
- Analyse de la diction (prononciation, fluidité, intonation)
- Recommandations d'exercices personnalisés
- Exercices guidés de pratique
